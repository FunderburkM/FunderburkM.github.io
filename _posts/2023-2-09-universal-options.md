# Universal Options

[Unreal Marketplace Link](https://www.unrealengine.com/marketplace/en-US/product/universal-options)  
[Documentation](https://github.com/FunderburkM/CMEngineOptionsDocs)  

![Featured Image](/assets/images/per-post/Universal-Options/FeaturedImage-v2.png)  

Universal Options is an Unreal Engine plugin available for Unreal Engine 4 (.24-.27) and Unreal Engine 5 (.0+). Powered by gameplay tags, json serialization, and data-oriented systems, its focus is providing an ample framework for developers to manage and extend the settings for their projects.  

Control and Expand:  

* Graphic Scalibility, Resolution, Active Monitor, and other dynamic settings  
* Audio Classes, Mixes, Levels, and Active Output Device  
* Standard Input Bindings, profiles, and dynamic switching  
* Enhanced Input Bindings, profiles, dynamic switching, and global data caching  
* Expandable Game Settings. Create and expand as many game setting configurations as you need
